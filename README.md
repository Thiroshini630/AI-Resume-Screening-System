# AI Resume Screening System

## Overview
This project is an AI-based resume ranking system using Natural Language Processing (NLP). 
It automatically compares resumes with job descriptions and ranks candidates based on similarity scores.

## Technologies Used
- Python
- Scikit-learn
- TF-IDF Vectorization
- Cosine Similarity

## Features
- Automated resume ranking
- Reduced manual screening effort
- NLP-based semantic matching

## Future Improvements
- Integration with ATS systems
- Deep Learning based semantic analysis
- Web-based interface
